## Go & Hugo 설치

Chocolatey 설치가 되었다는 가정하

1. GoLang 설치

```bash
choco install golang
```

2. hugo 설치

```bash
choco install hugo
```

3. hugo-extended 설치

```bash
choco install hugo-extended
```
