- UI : USER INTERFACE
- UI는 사용자가 제품/서비스를 사용할 때, 마주하게 되는 면
- 사용자가 제품/서비스와 상호작용할 수 있도록 만들어진 매개체
- UI 디자인은 폰트, 칼라, 레이아웃과 같이 사용자가 마주하게될 시각적인 디자인
