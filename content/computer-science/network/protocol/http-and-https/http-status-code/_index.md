-   응답은 5개의 그룹으로 나뉘어 진다
    -   정보를 제공하는 응답 (1xx)
    -   성공적인 응답 (2xx)
    -   리다이렉트 (3xx)
    -   클라이언트 에러 (4xx)
    -   서버 에러 (5xx)
